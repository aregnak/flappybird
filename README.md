## Flappy Bird in SFML v3.0 to learn more C++

### How to run:
#### Linux: 
```bash
# clone repository
git clone https://github.com/aregnak/flappybird.git
cd flappybird
```
Compiling and running:
```bash
# build using make
make 
# OR
make compile
#run with
./flappy
# OR
make run

# build using shell script
./build.sh
#run with
./flappy
```
